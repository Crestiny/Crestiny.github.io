¶1. 图搜索技术

搜索算法分类：

  • Blind Search Strategies
    • Breadth-First search
    • Depth-First search
    • Uniform-Cost search
  • Heuristic Search Strategies
    • Best-first search/Ordered Search
    • Greedy Search
    • Algorithm A
    • Algorithm A*
    • Hill climbing

注意：树的搜索可以不需要closed表，因为并不存在回头的情况

算法总结：

算法 数据结构 估价函数
BFS FIFO队列 无
DFS LIFO队列（栈） 无
UCS 优先队列（每次扩展cost最小的点） f(n)=g(n)
贪婪 优先队列 f(n)=h(n)
A 优先队列 f(n)=g(n)+h(n)
A* 优先队列 f*(n)=g*(n)+h*(n)
